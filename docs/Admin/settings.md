# API админ-панели: Настройки сайта

**Базовый URL:** `/api/admin/settings`

**Авторизация:** все эндпоинты требуют авторизации администратора (`AdminGuard`). Запросы авторизуются через httpOnly cookie `adminToken`, которую браузер отправляет автоматически (заголовок `Authorization` также принимается для обратной совместимости).

Универсальное key-value хранилище настроек. Значения автоматически распаршиваются из JSON (числа, строки, объекты, массивы, булево).

---

## Стандартные ключи настроек

| Ключ | Назначение | Значение по умолчанию |
|------|------------|-----------------------|
| `delivery_price_cdek` | Fallback для CDEK самовывоза, если CDEK API недоступен. По умолчанию доставка считается реально через CDEK API (по городу + весу) | `500` |
| `delivery_price_standard` | Цена почтовой доставки (фикс, ₽) — для стран без CDEK | `800` |
| `low_stock_threshold` | Порог низкого остатка | `5` |
| `site_name` | Название сайта | `"Saliy Clothes"` |
| `contact_email` | Контактный email | `"info@saliyclothes.com"` |

Список расширяется — можно добавлять любые произвольные ключи.

---

## 1. Получить все настройки

**GET** `/api/admin/settings`

Ответ — массив объектов настроек с полями `key`, `value`, `description`, `updatedAt`.

---

## 2. Получить настройку

**GET** `/api/admin/settings/:key`

| Параметр пути | Тип | Описание |
|---------------|-----|----------|
| `key` | string | Ключ настройки |

Ответ — объект с полями `key`, `value`, `description`, `updatedAt`.

---

## 3. Создать / обновить настройку

**PUT** `/api/admin/settings/:key`

Upsert — создаёт, если ключа нет; обновляет, если есть.

| Поле | Тип | Обязательное | Описание |
|------|-----|--------------|----------|
| `value` | любой JSON-тип | Да | Значение (число, строка, объект, массив, булево) |
| `description` | string | Нет | Описание настройки |

---

## 4. Удалить настройку

**DELETE** `/api/admin/settings/:key`

| Параметр пути | Тип | Описание |
|---------------|-----|----------|
| `key` | string | Ключ настройки |

Ответ: `success: true`.

---

## Важно: цены доставки

Поведение не фикс-цена для всех:

- `CDEK_PICKUP` → бэк запрашивает CDEK API (`/v2/calculator/tarifflist`) с городом получателя и весом товаров → берёт реальный тариф «склад-склад». Если API недоступен / `cdekCityCode` не передан / товар без `weight` — fallback на `delivery_price_cdek`.
- `STANDARD` → фикс из `delivery_price_standard` (CDEK не покрывает все страны).

`delivery_price_cdek` играет роль страховки, а не основной цены. Изменение применяется без редеплоя — бэк читает настройку при каждом расчёте.

См. [docs/shop/delivery.md](../shop/delivery.md), раздел «Расчёт стоимости», для деталей механики.
