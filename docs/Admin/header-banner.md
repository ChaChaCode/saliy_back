# Текстовый баннер шапки

Одна глобальная бегущая строка (текст), которую фронт выводит в шапку сайта по кругу. Хранится в настройках сайта под ключом `header_banner`. Состоит из текста и флага видимости.

---

## Структура баннера

| Поле | Тип | Описание |
|------|-----|----------|
| `text` | string | Текст бегущей строки (до 500 символов) |
| `enabled` | boolean | Показывать ли баннер в шапке |

---

## Эндпоинты

### Получить баннер (публичный)

**GET** `/api/header-banner`

- Авторизация не требуется — используется витриной для вывода строки в шапке.
- Если баннер не задан, возвращается `{ text: "", enabled: false }`.

**Поля ответа:**

| Поле | Тип | Описание |
|------|-----|----------|
| `text` | string | Текущий текст баннера |
| `enabled` | boolean | Активен ли баннер |

Фронт показывает бегущую строку, только если `enabled = true` и `text` не пустой.

---

### Получить баннер (админ)

**GET** `/api/admin/header-banner`

- Авторизация: требуется (администратор).
- Возвращает те же поля, что и публичный эндпоинт. Используется в админке для предзаполнения формы.

---

### Задать баннер (админ)

**PUT** `/api/admin/header-banner`

- Авторизация: требуется (администратор).

**Поля запроса:**

| Поле | Тип | Обязательное | Описание |
|------|-----|--------------|----------|
| `text` | string | да | Текст баннера (до 500 символов). Пробелы по краям обрезаются. |
| `enabled` | boolean | нет | Показывать ли баннер. По умолчанию `true`. |

**Ответ:** сохранённый баннер (`text`, `enabled`).

Чтобы скрыть баннер — передать `enabled: false` (текст при этом можно сохранить).
