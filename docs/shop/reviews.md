# API отзывов (публичное)

**Базовый URL:** `/api/reviews`

Отзывы оставляют как гости, так и авторизованные пользователи. Все отзывы проходят модерацию — публикуются только после одобрения админом.

---

## 1. Оставить отзыв

**POST** `/api/reviews`

**Авторизация:** опциональна (если есть — привязывается к `userId`)

### Request:
```json
{
  "productId": 20,
  "authorName": "Иван П.",
  "rating": 5,
  "text": "Отличное качество, размер подошёл идеально. Рекомендую!"
}
```

### Поля:
- **productId** — ID товара (обязательно)
- **authorName** — имя автора (обязательно, до 100 символов)
- **rating** — оценка 1-5 (обязательно)
- **text** — текст отзыва (опционально, до 2000 символов)

### Response:
```json
{
  "id": "review-uuid",
  "productId": 20,
  "userId": "user-uuid-or-null",
  "authorName": "Иван П.",
  "rating": 5,
  "text": "Отличное качество...",
  "status": "PENDING",
  "createdAt": "2026-04-11T12:00:00.000Z",
  "updatedAt": "2026-04-11T12:00:00.000Z"
}
```

> Отзыв **не виден на сайте**, пока админ его не одобрит (`status: APPROVED`).

### Ошибки:
- `400` — невалидный рейтинг (вне 1-5)
- `400` — авторизованный пользователь уже оставил отзыв на этот товар
- `404` — товар не найден

---

## 2. Получить одобренные отзывы товара

**GET** `/api/reviews/product/:productId`

Возвращает **только одобренные** отзывы + средний рейтинг.

### Response:
```json
{
  "reviews": [
    {
      "id": "review-uuid",
      "authorName": "Иван П.",
      "rating": 5,
      "text": "Отличное качество...",
      "createdAt": "2026-04-11T12:00:00.000Z"
    }
  ],
  "averageRating": 4.7,
  "totalReviews": 42
}
```

### Поля:
- **averageRating** — средняя оценка (округлена до 0.1)
- **totalReviews** — количество одобренных отзывов
