# Текстовый баннер шапки

Глобальная бегущая строка в шапке сайта. Текст задаётся администратором, витрина читает его и крутит по кругу.

Глобальный префикс приложения — `/api`.

---

## Получить баннер шапки

**GET** `/api/header-banner`

Авторизация не требуется.

**Поля ответа:**

| Поле | Тип | Описание |
|------|-----|----------|
| `text` | string | Текст бегущей строки |
| `enabled` | boolean | Показывать ли баннер |

Если баннер не задан — возвращается `{ text: "", enabled: false }`.

---

## Поведение на витрине

- Показывать бегущую строку в шапке, только если `enabled = true` и `text` не пустой.
- Текст крутится по кругу (бегущая строка / marquee).
- Управление текстом и видимостью — на стороне администратора.
